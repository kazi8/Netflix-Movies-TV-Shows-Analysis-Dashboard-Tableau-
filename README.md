An interactive dashboard that breaks down Netflix movies and TV shows by genre, rating, country, and release year to uncover global content patterns.

Interactive Data Visualization Project

This repository contains a Netflix Data Dashboard built in Tableau, following a guided tutorial from YouTube. The goal of this project was to practice data visualization, dashboard design, and storytelling with data using a real‑world dataset of Netflix titles.

Although the dashboard was created by following a tutorial, the project demonstrates my ability to:

•  Work with Tableau worksheets and dashboards
•  Build multi‑chart layouts
•  Apply filters, parameters, and interactivity
•  Design clean, modern visualizations
•  Understand data storytelling concepts
